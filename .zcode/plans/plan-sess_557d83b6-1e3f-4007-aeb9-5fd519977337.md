## Goal
RTE di Second Brain makin panjang ke bawah — flex fill kolom editor penuh. No cap.

## Single file change
`frontend/src/lib/components/SecondBrainNotes.svelte` — tambah 2 scoped CSS rules di block `<style>`:

```css
/* RTE: isi kolom editor penuh di Second Brain saja.
   .sb-editor class cuma ada di sini → ProjectDetails/NotesEditor asli aman. */
:global(.sb-editor .ne-editor-host) { flex:1 1 auto; min-height:0; display:flex; }
:global(.sb-editor .ne-editor-host .ne-textarea) { flex:1 1 auto; min-height:280px; max-height:none; }
```

## Apa yang TIDAK disentuh
- `NotesEditor.svelte` (shared component, max-height:300px di line 1185 tetap utuh)
- `ProjectDetails.svelte` (pakai NotesEditor juga — harus tetap 300px cap)
- `styles.css`, komponen lain, backend, bridge
- Hanya CSS block di SecondBrainNotes.svelte

## Why scoped aman
- `.sb-editor` class cuma ada di `SecondBrainNotes.svelte:928` → selector `:global(.sb-editor .ne-editor-host ...)` cuma match Second Brain
- Project Details render NotesEditor via wrapper lain (class berbeda) → tidak kena override
- `flex:1 1 auto` di textarea + `display:flex` di host = textarea ngisi ruang tersisa di `.sb-editor` (flex col) setelah toolbar
- `max-height:none` override cap 300px yang shared di NotesEditor
- `min-height:280px` jaga window kecil tetap reasonable
- `resize:vertical` (dari NotesEditor) tetap aktif kalau user mau kecilkan manual
