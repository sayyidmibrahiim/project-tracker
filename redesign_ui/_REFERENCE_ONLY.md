# Reference Only

PyQt6 files in this directory are UX/function reference only. Use them to understand layout, interactions, menu behavior, and user flows.

Do not run or integrate PyQt6 files into the production application.

Old static HTML frontend files are legacy/reference only. They are not migrated production screens.

Production frontend target is Svelte + TypeScript + Vite, with Tailwind CSS bundled through the frontend build.
