Project Tracker DBS — Help Guide

Use this application to track local-first DBS CR delivery work. Project folders are the source of truth for project existence and state.

Dashboard

Use status tabs to filter projects. Drone Ticket and CR Number fields auto-commit when focus leaves the field.

Project Details

Edit project metadata, sub projects, files, notes, and activity history from one workspace.

Automations

Configure Outlook send/download, Teams messages, and Reminder rules without leaving local-first constraints.
