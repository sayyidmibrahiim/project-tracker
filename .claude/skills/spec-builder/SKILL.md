# Spec Builder

Use before new menu work or significant redesign.

Create `_specs/SPEC-[menu].md` with:

- Goal and non-goals
- PRD sections used
- DESIGN.md references
- Wireframe logic in text
- Components and all states
- User flow
- Data source and bridge methods
- Shared elements and propagation list
- Edge cases
- Manual checklist draft

Keep scope to one menu. Do not implement code.
